# 🎸 BANDme!

音楽仲間（個人 / バンド）を見つけてつながれる、SNS風のWebアプリです。  
投稿フィードで募集・近況を発信し、ジャンル/楽器/タグで検索。気になる相手とはDMで連絡できます。  
📱 **スマホ版 / 💻 パソコン版どちらでも快適に使えるレスポンシブ対応**です。

## 🎮 主な機能

- **レスポンシブUI（スマホ/PC対応）**
  - 画面幅に応じてレイアウトが切り替わる設計
  - スマホではフィルターをアイコンで開閉して操作しやすいUI
  - モーダル/チャット画面もスマホで全画面最適化

- **投稿フィード（画像/動画対応）**
  - キャプション・ジャンル・自分の楽器・相手の楽器・タグを登録
  - 投稿の編集 / 削除（本人のみ）
  - 「自分の投稿だけ表示」トグル

- **検索 / フィルター**
  - ロール：個人のみ / バンドのみ / 全員
  - ジャンル・楽器・タグ（複数）・キーワード検索（投稿文/タグ/ユーザー名）

- **プロフィール**
  - ユーザー名・bio（自己紹介）編集
  - アイコン画像アップロード
  - ショーケース（画像/動画をまとめて掲載）

- **フォロー**
  - フォロー / アンフォロー
  - フォロワー / フォロー一覧

- **チャット（DM）**
  - 1対1の会話
  - 既読管理・未読表示
  - 自分のメッセージ削除 / 会話の非表示（自分側のみ）
